How to read the map

Use this page as a provenance lens, not a popularity chart. Origin rows are shown first because they are more interpretable than CDN-resolved locations for source tracing.

  • Cluster density shows where source infrastructure concentrates.
  • Popup ISP labels help separate institutional hosting from generic delivery layers.
  • CDN rows are intentionally hidden by default for cleaner trust signals.
