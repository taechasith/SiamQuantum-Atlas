Topic axis

Topic labels are drawn from `entities.area`, with quantum-domain fallback only when no finer topic text exists in the row.

Production axis

Production labels are normalized from `media_format_detail` first, then backed by media-format, content-type, and production-type fallbacks so article, podcast, vlog, course, paper, film, game, and related forms can surface cleanly.

Reading rule

Treat visually strong lines as hypotheses first. The validation cards underneath each chart show whether the current corpus supports a statistically meaningful split.

Scientific basis
SciPy, pymannkendall, Chart.js, and 3d-force-graph drive the calculations and the views.
Engagement metric
Each point is a bootstrap geometric mean on log1p view-count engagement, which is more robust than raw arithmetic means for skewed web data.
3D graph logic
Each 3D graph links years to categories using observed counts, then colors communities from weighted graph structure instead of arbitrary styling.
